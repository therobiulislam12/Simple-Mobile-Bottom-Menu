Simple Mobile Bottom Menu plugin for smooth navigation with customizable styles.

== Description ==

The Simple Mobile Bottom Menu plugin allows you to create a custom bottom navigation menu tailored for mobile devices. Easily add menu items with custom images through an intuitive admin interface. 

This plugin is perfect for enhancing mobile user experience by providing a sleek and customizable bottom menu.

== Features ===

1. Register a custom bottom menu for mobile devices.
2. Add custom images to menu items via a user-friendly admin interface.
3. Seamless integration with WordPress's native menu management system.
4. Lightweight and optimized for mobile performance.
